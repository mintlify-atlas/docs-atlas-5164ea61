<Accordion title="DateString">

A date formatted as `year-month-day`.

```typescript
type DateString = `${number}-${number}-${number}`;
```


<Accordion title="Filter">

A filter object used to narrow analytics queries.

```typescript

    parameter: FilterParameter;
    type: FilterType;
    value: Array<string>;
}
```

<ResponseField name="parameter" type="FilterParameter" required>
  The dimension to filter on. See [FilterParameter](#filterparameter).
</ResponseField>

<ResponseField name="type" type="FilterType" required>
  The comparison operator. See [FilterType](#filtertype).
</ResponseField>

<ResponseField name="value" type="Array<string>" required>

</ResponseField>

**Example:**


### `handleRejection`

A utility for catching rejected promises from SDK methods. Guarantees that the error passed to your callback is always a typed [`ErrorResponse`](#errorresponse), regardless of what was actually thrown.

```typescript
function handleRejection(
    callback: (error: ErrorResponse) => void | Promise<void>
): (reason: any) => void
```

**Example:**

```typescript
import { Rest, handleRejection } from 'rybbit.ts';

const client = new Rest({ domain: '...', siteId: 123, apiKey: '...' });

client
    .getUsers()
    .then((users) => console.log(users))
    .catch(handleRejection((err) => {
        console.error(`[${err.status}] ${err.error}`);
    }));
```
