# rybbit.ts

A TypeScript wrapper for the [rybbit.com](https://rybbit.com) analytics API. Built for server-side runtimes such as Node.js, Bun, and Deno — so you can track events, query analytics data, and manage your sites directly from your backend.

## What is rybbit.ts?

rybbit.ts is an npm package that wraps the rybbit.com REST API with full TypeScript support. It gives you a clean, typed interface to:

- **Track events server-side** — pageviews and custom events with custom properties
- **Query analytics** — overview metrics, time-series data, metric breakdowns by parameter
- **Manage sites and organizations** — create sites, manage members, update configuration
- **Analyze user behavior** — sessions, user profiles, journeys, retention cohorts
- **Monitor errors** — error names, occurrences, and time-series error counts
- **Define and track goals** — path and event-based conversion goals
- **Build funnels** — multi-step conversion funnel analysis
- **Measure performance** — Core Web Vitals overview, time-series, and dimension breakdowns

## Quick example

```typescript
import rybbit from 'rybbit.ts';

const analytics = new rybbit({
    domain: 'https://example.rybbit.com',
    siteId: 1,
    apiKey: process.env.RYBBIT_API_KEY
});

// Track a custom event server-side
const result = await analytics.track({
    type: 'custom_event',
    ip_address: '8.8.8.8',
    event_name: 'user_signup',
    properties: { plan: 'pro' }
});

// Result: { success: true }
```
