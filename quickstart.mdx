This guide walks you from a fresh install to a working analytics integration.

<Steps>

<Step title="Import and instantiate the client">

Import the default export and create a client instance using `RestConfig`. You need your rybbit domain, site ID, and API key.

```typescript
import rybbit from 'rybbit.ts';

const analytics = new rybbit({
    domain: 'https://example.rybbit.com',
    siteId: 1,
    apiKey: process.env.RYBBIT_API_KEY
});
```

<Tip>
  Store your API key in an environment variable and never commit it to source
  control. Use a `.env` file or your hosting provider's secrets manager.
</Tip>

The `RestConfig` interface accepts:

| Field     | Type     | Description                             |
| --------- | -------- | --------------------------------------- |
| `domain`  | `string` | The base URL of your rybbit instance    |
| `siteId`  | `number` | The numeric ID of the site to track     |
| `apiKey`  | `string` | Your rybbit API key                     |

</Step>

<Step title="Track a custom event">

Use `analytics.track()` to send a `custom_event` from your server. Pass an `ip_address` so rybbit can attribute the event to the correct location.

```typescript
const result = await analytics.track({
    type: 'custom_event',
    ip_address: '8.8.8.8',
    event_name: 'user_signup',
    properties: { plan: 'pro', referral: 'homepage' }
});

// result: { success: true }
```

The `properties` field accepts any `Record<string, any>` — pass whatever metadata is meaningful for the event.

</Step>

<Step title="Track a pageview">

To record a server-rendered pageview, set `type` to `'pageview'` and include the path and hostname:

```typescript
const result = await analytics.track({
    type: 'pageview',
    hostname: 'example.com',
    pathname: '/pricing',
    ip_address: request.headers.get('x-forwarded-for') ?? '127.0.0.1',
    user_agent: request.headers.get('user-agent') ?? undefined,
    referrer: request.headers.get('referer') ?? undefined
});

// result: { success: true }
```

<Note>
  Forward the real `ip_address` and `user_agent` from the incoming request so
  rybbit can correctly resolve the visitor's location, browser, and device.
</Note>

</Step>

<Step title="Handle errors">

All methods return a rejected promise on failure with an `ErrorResponse` object. Use the `handleRejection` utility to get typed errors in your `.catch()` handler:

```typescript
import rybbit, { handleRejection } from 'rybbit.ts';

const result = await analytics
    .track({
        type: 'custom_event',
        event_name: 'checkout_complete'
    })
    .catch(
        handleRejection((error) => {
            // error is typed as ErrorResponse
            console.error(`[${error.status}] ${error.error}`);
        })
    );
```

The `ErrorResponse` status codes:

| Status | Meaning                               |
| ------ | ------------------------------------- |
| `-1`   | Request failed to send                |
| `-2`   | Response was not valid JSON           |
| `-3`   | Unknown response                      |
| `XXX`  | HTTP status code returned by server   |

Without `handleRejection`, the raw rejection reason may not be a typed `ErrorResponse`. The utility normalizes it so your callback always receives the correct shape.

</Step>

<Step title="Query analytics data">

Once events are flowing, you can read analytics back using the query methods. Here is an example using `getOverview` to fetch high-level metrics for a date range:

```typescript
import { toDateString } from 'rybbit.ts';

const today = toDateString(new Date());

const overview = await analytics
    .getOverview({
        startDate: '2024-01-01',
        endDate: today,
        timezone: 'UTC'
    })
    .catch(
        handleRejection((error) => {
            console.error(`Failed to fetch overview: ${error.error}`);
        })
    );

console.log(overview);
```

The `toDateString` helper converts a JavaScript `Date` to the `YYYY-MM-DD` format (`DateString`) required by all date parameters.

</Step>

</Steps>

## Full example

Here is everything together in a single file:

```typescript
import rybbit, { handleRejection, toDateString } from 'rybbit.ts';

const analytics = new rybbit({
    domain: 'https://example.rybbit.com',
    siteId: 1,
    apiKey: process.env.RYBBIT_API_KEY
});

// Track a custom event
await analytics
    .track({
        type: 'custom_event',
        ip_address: '8.8.8.8',
        event_name: 'user_signup',
        properties: { plan: 'pro' }
    })
    .catch(handleRejection((error) => console.error(error)));

// Track a pageview
await analytics
    .track({
        type: 'pageview',
        hostname: 'example.com',
        pathname: '/home'
    })
    .catch(handleRejection((error) => console.error(error)));

// Query overview metrics
const overview = await analytics
    .getOverview({ startDate: '2024-01-01', endDate: toDateString(new Date()) })
    .catch(handleRejection((error) => console.error(error)));

console.log(overview);
```
